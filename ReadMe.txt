To Run the visualization and preprocessing python code you have to follow these steps:

1.Open jupyter notebook
2.Install the required libraries using "Pip install" command if the required libraries are not installed in your computer
3.Upload the "New_CSV" file into the jupyter notebook
4.Check whether you have installed all the libraries for your corresponding pip version
5.Check whether the sklearn installed path is added to sys path to avoid  any error
6.Run the code to get the output
